# Headphones – ALX Front-End (HTML & CSS)

## Project Overview

This project is part of Week 7 of the ALX Front-End Development program.
The goal is to recreate a fully responsive landing page from scratch using only HTML and CSS, based on a Figma design created by Nicolas Philippot (UI/UX Designer).

The final implementation matches the desktop, tablet, and mobile designs while respecting all technical constraints.

---

## Objectives

- Reproduce the Figma design accurately
- Use semantic HTML
- Apply modern CSS (Flexbox)
- Ensure accessibility and responsiveness
- Follow ALX restrictions strictly

---

## Technologies Used

- HTML5
- CSS3
- Responsive Design
- Accessibility best practices

No JavaScript  
No external CSS frameworks

## Responsive Behavior

- Desktop: above 768px
- Tablet: 768px and below
- Mobile: 480px and below

At 480px or less, the layout switches to the mobile version.

---

## Interactions

- Link hover/active color: #FF6565
- Button hover/active: opacity 0.9
